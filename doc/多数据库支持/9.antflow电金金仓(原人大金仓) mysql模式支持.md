# antflow kingbase MySql模式

创建docker镜像方式同第八篇,只需要把模式改为Mysql即可


```
docker run -idt \
--name KingBase \
--privileged \
--restart=always \
-p 54321:54321 \
-e DB_MODE=mysql\
-v /data/kingbase/data:/home/kingbase/mysql/data \
kingbase_v8:v1
```

> 注意,改了模式,挂载的目录也要做相应调整.

连接数据库这里也不再赘述了.
