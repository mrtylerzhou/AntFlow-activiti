antflow kingbase PG模式

PG模式直接使用PG仓库里的PG脚本即可，完全兼容。但是驱动最好还是要使用kingbase官方驱动。

所有信息查看第八章kingbase oracle模式支持，只需要改掉模式即可

```
docker run -idt \
--name KingBase \
--privileged \
--restart=always \
-p 54321:54321 \
-e DB_MODE=pg\
-v /data/kingbase/data:/home/kingbase/pg/data \
kingbase_v8:v1

```
